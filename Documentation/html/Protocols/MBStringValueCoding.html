MBStringValueCoding Protocol Reference

Conforms to NSObject
Declared in MBVariableDeclaration.h

Overview

Classes that adopt this protocol gain the ability to be stored in the NSUserDefaults as strings. This allows native objects to be stored using the variable’s userDefaultsName mechanism.

Class Methods

fromStringValue:

Constructs and returns a new, autoreleased instance of the receiver. The returned object’s internal state will be set based on the value of the passed-in string.
