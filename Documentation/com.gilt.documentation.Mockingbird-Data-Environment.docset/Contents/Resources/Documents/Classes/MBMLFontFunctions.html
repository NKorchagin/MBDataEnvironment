Overview

A class containing implementations for MBML functions that return information about the fonts available at runtine.

See MBMLFunction for more information on MBML functions and how they’re used.

fontFamilyNames

An MBML function implementation that returns an array of the available UIFont family names.

+ (id)fontFamilyNames

Return Value

The names of the available font families.

Discussion

This function takes no parameters.

MBML Example

The expression:

^fontFamilyNames()

would return the names of all font families available on the device on which the expression is evaluated.

fontNamesForFamilyName:

An MBML function implementation that returns an array of the available UIFont family names.

+ (id)fontNamesForFamilyName:(NSString *)input

Parameters

input

The function’s input parameter.

Return Value

The names of the fonts available in the given font family.

Discussion

This function takes a single expression as input, the name of the font family whose font names are sought.

MBML Example

On a first-generation iPad, the expression:

^fontNamesForFamilyName(Helvetica)

would return an array containing the following font names:

  • Helvetica-LightOblique
  • Helvetica
  • Helvetica-Oblique
  • Helvetica-BoldOblique
  • Helvetica-Bold
  • Helvetica-Light

fontWithNameAndSize:

An MBML function implementation that returns a UIFont instance for the given font name and size.

+ (id)fontWithNameAndSize:(NSArray *)params

Parameters

params

The function’s input parameters.

Return Value

The requested UIFont.

Discussion

This function takes a two parameters as input:

  • The font name, a string expression that specifies the name of the font requested

  • The font size, a numeric expression indicating the size of the font to return

MBML Example

^fontWithNameAndSize(Helvetica|14)

would return a UIFont instance for the font named “Helvetica” at 14-point size.

sizeOfTextWithFont:

An MBML function implementation that measures the size of text in a given font.

+ (id)sizeOfTextWithFont:(NSArray *)params

Parameters

params

The function’s input parameters.

Return Value

The requested UIFont.

Discussion

This function takes a two parameters as input:

  • The text, a string expression specifying the text to measure

  • The font, an object expression that yields a UIFont instance

The function returns a string containing the size of the specified text in “width,height” format.

MBML Example

^sizeOfTextWithFont(This is my text to measure|^fontWithNameAndSize(Helvetica|14))

would return a UIFont instance for the font named “Helvetica” at 14-point size.
