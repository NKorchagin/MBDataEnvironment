containsString:

Determines if the string specified by the first expression parameter contains the string indicated by the second expression parameter.

+ (id)containsString:(NSArray *)params

Parameters

params
  • the input parameters

Return Value

An NSNumber containing the boolean result.

Discussion

This function accepts two parameters:

The first parameter a string expression that will be tested to see if it contains the substring.

The second parameter is a string expression containing the substring.

Template example:

^containsString($dayOfWeek|ues)

The template expression above returns YES if the string value of the expression $dayOfWeek contains the string “ues”.

q:

Allows for quoting a string in the template language. This is required, for example, to preserve space characters at the beginning or end of a string literal. (Normally, the expression evaluator ignores whitespace at the beginning or end of string literals.)

+ (id)q:(NSString *)toQuote

Parameters

toQuote

the string

Return Value

toQuote

Discussion

This function accepts one parameter, the string to be quoted.

Template example:

^q(   -- Please leave spaces intact --   )

The template expression above quotes the string so that the space characters at the beginning and end of the string are preserved.
